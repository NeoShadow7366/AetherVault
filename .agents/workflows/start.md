Let's begin a new session.

1. Locate and read the most recent handoff summary or session log (e.g., session-summary.md, today's date-based log, or the latest entry in project logs).

2. Summarize the current overall project state, including:
   - Key accomplishments from previous sessions
   - Open blockers, questions, or TODOs
   - Recommended starting points or focus areas

3. Check and briefly report on the status of relevant project documents, agent instructions, Rules, and Skills.

4. Load any important context from the last session so we can continue without repetition.

5. Ask me what I want to focus on in this session, or suggest 2-3 productive next steps based on the current state.

Make everything clear and structured with bullet points where helpful. 
Once loaded, end with: "Session started successfully. Ready to work. Current project state loaded."
